# Faiz Ahmad – Java Full Stack Developer Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## 📁 Folder Structure

```
Faiz_portfolio/
├── index.html          # Main HTML file (all sections)
├── css/
│   └── style.css       # All styles + dark/light theme + responsive
├── js/
│   └── main.js         # Theme toggle, typed effect, scroll, form
├── assets/
│   └── Faiz_Ahmad_Resume.pdf   # Place your resume PDF here
└── README.md
```

## 🚀 How to Run

### Option 1 – Open directly (simplest)
1. Double-click `index.html` — opens in your default browser.

### Option 2 – VS Code Live Server (recommended)
1. Open the `Faiz_portfolio` folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.

### Option 3 – Python local server
```bash
cd Faiz_portfolio
python3 -m http.server 3000
# Open http://localhost:3000
```

## ✨ Features

- Dark / Light theme toggle (persisted in localStorage)
- Typed text animation in hero section
- Smooth scroll navigation with active link highlighting
- Fade-in animations on scroll (IntersectionObserver)
- Project cards with hover effects
- Fully responsive (mobile + tablet + desktop)
- Download Resume button
- Scroll-to-top button

## 🎨 Customization

1. Replace `faiz@example.com` with your real email in `index.html`
2. Update GitHub/LinkedIn URLs throughout `index.html`
3. Add your resume PDF to `assets/Faiz_Ahmad_Resume.pdf`
4. Update education details, project descriptions, and certifications as needed
